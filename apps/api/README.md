# CIC CORE_V2

O CORE_V2 tem como objetivo integrar os persionamentos e suas respectivas funcionliades além da integração com o login exerto via keykcloak.

<br />
<br />

## Historico

### Corporação
- Criação Corporação
- Edição Corporação

### Unidade
- Criação Unidade
- Edição Unidade
- Adição/remoção de unidade do grupo

### Unidade Gerencial
- Criação Unidade Gerencial
- Edição Unidade Gerencial

### Grupo
- Criação Grupo
- Edição Grupo
- Adição/remoção de unidade ao grupo
- Associação/desassociação de users ao grupo

### Empresa
- Criação Empresa
- Edição Empresa
- Ativação/desativação Empresa
- Adição/remoção de unidade ao grupo
- Associação de grupo/unidade à empresa
- Associação/desassociação de users ao grupo

### User
- Ativação/desativação User
- User autor da ação (todos os casos)
- Associação/desassociação de users ao grupo
